Assess in 60 seconds

  • Fish school & wander organically; none identical
  • Press T → water darkens, predator, scatter
  • Press S → sonar ring, curious fish approach
  • AI beam periodically scans & clears a fish
  • Type debug → FPS steady, RSS memory flat
  • Palette stays violet/cyan; red only on threat
